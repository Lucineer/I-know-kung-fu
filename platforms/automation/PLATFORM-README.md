# Automation Platforms

> **n8n, Zapier, and Gumloop Integration Guide**

## Platform Overview

Automation platforms allow users to connect AI agents to their existing workflows. These are high-value B2B opportunities.

### n8n
- Self-hosted workflow automation
- Growing developer community
- Strong AI agent support
- Fair code license

### Zapier
- Largest automation platform
- 5,000+ integrations
- Consumer and business users
- Usage-based pricing

### Gumloop
- AI-native automation
- VC-backed ($20M raised)
- Growing fast
- Agent-focused

## Strategy

Build reusable workflow templates that showcase your agents:

| Platform | Strategy | Price Point |
|----------|----------|-------------|
| n8n | Open source templates + paid support | $99-299/support |
| Zapier | Premium app with usage pricing | $0.05-0.50/action |
| Gumloop | Agent marketplace listing | Credit-based |
